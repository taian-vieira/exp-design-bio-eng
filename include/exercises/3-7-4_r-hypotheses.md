Question
========

*Given the maximal heart rate and age data shown in @fig-3-7-2, what would be the null hypothesis and the alternative hypothesis for the correlation coefficient?*

Answerlist
----------
* $H_0: \rho = 0, \, H_1: \rho < 0$
* $H_0: r = 0, \, H_1: \rho < 0$
* $H_0: r = 0, \, H_1: \rho \ne 0$
* $H_0: \rho = 0, \, H_1: \rho \ne 0$
* $H_0: r = 0, \, H_1: r < 0$

Solution
========

The null and alternative hypotheses are framed according to the population correlation coefficient $\rho$. This leaves only two valid options to be considered.

As the scatterplot shows a negative relationship between maximal heart rate and age, the alternative hypothesis to be tested would be that $\rho < 0$.
