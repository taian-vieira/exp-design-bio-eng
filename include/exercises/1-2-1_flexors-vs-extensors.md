Question
========

*Can we readily tell which muscles are active and when?*

Answerlist
----------
* *only flexors ($\uparrow$ and $\downarrow$)*
* *only extensors ($\uparrow$ and $\downarrow$)*
* *flexors ($\uparrow$), extensors ($\downarrow$)*
* *flexors ($\downarrow$), extensors ($\uparrow$)*
* *it is not possible to say*

Solution
========

It is not possible to say, given no information is provided about the gravity vector with respect to the segments shown.

If gravity vector was in the downward direction in the figure, the correct answer would be that elbow flexor muscles are contracting both in the up and down phase.
In the up phase they would be producing positive work, moving the limb against gravity. In the down phase they would be producing negative work, obsorbing gravity work as the limb is moved down.
