Question
========

*A statistical power of $25\%$ indicates that:*

Answerlist
----------

* there is a $25\%$ probability of rejecting $H_0$ when it is true
* there is a $25\%$ probability of not rejecting $H_0$ when it is false
* the statistical test is unlikely to detect a true effect
* the sample size is sufficient to detect the expected effect
* none of the above

Solution
========

Statistical power is defined as the probability of rejecting $H_0$ when it is false.

A power of $25\%$ therefore means that, even if a true effect exists, the test will correctly detect it only $25\%$ of the time. Equivalently, there is a $75\%$ probability of failing to detect the effect (Type II error).

This indicates that the test has low sensitivity and is unlikely to detect a true difference between groups.

Therefore, the correct answer is that the statistical test is unlikely to detect a true effect.
